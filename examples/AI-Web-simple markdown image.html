# Enhanced Markdown Image Guidelines

## Syntax

Standard: `![Alt text](url)`
Enhanced: `![Alt text|width|align|caption](url)`

Parameters (all optional after alt text):

1. **Alt text** - Fallback description
1. **Width** - Pixels (200) or percent (40%)
1. **Align** - `center` only (left is default)
1. **Caption** - Text displayed below image

## Examples

**Basic image**
`![Bee](url)` ‚Üí Left-aligned, no width constraint

**Fixed width**
`![Bee|200](url)` ‚Üí 200px wide

**Percentage width**
`![Bee|40%](url)` ‚Üí 40% of container

**Centered**
`![Bee|200|center](url)` ‚Üí Centered, 200px wide

**With caption**
`![Bee|200|center|Our friendly bee](url)` ‚Üí All features

**Caption only**
`![Bee|||Caption text](url)` ‚Üí Skip width/align parameters

## Key Features

- HTML-escaped captions and alt text for security
- Native lazy loading on all images
- All images wrapped in `
